Calculandum Machina - help

Description

This document is made to help with using our calculator. Here you will find everything to be able to do the math operations.

Layout

Buttons have different background colors for better orientation.
All number buttons plus dot are white.
Help and math functions are light green.
Clear and result buttons are darker green.

Table describing layout

Display 1
Display 2
help logarithm root
7 8 9 power factorial
4 5 6 multiplication division
1 2 3 addition subtraction
0 . clear result

Keyboard mapping

Every button can be substituated with key on keyboard.

"?/h/H" - help

"+" - addition
"-" - subtraction
"*" - multiplication
"/" - division
"p/P" - power
"r/R" - root
"f/F" - factorial
"l/L" - logarithm

"c/C" - clear
"return/enter" - result

Numbers are of course number keys.

How to use

Start the aplication.

Click on operands and operations you want to use.

Usage of different operations

With operations addition, subtraction, multiplication and division insert first number, then the operation, then second number and hit enter or result button.
With operation power first insert the number you want to be the base, then hit the power button, than insert second number that will be the exponent and hit result
With operation root first insert the number you want to be the base, then hit the root button, than insert second number that will be the index of the root.
With operations factorial and logarithm insert number and then the operation. Result will show straightaway.

You can also use keyboard for that.
On Display 1 you will see what you have written.
Result will show on Display 2.

To exit the aplication hit the red cross in top right corner.
