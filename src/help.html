Calculandum Machina - help

Description

This document is made to help with using our calculator. Here you will find everything to be able to do the math operations.

Layout

Buttons have different background colors for better orientation.
All number buttons plus dot are white.
Help and math functions are light green.
Clear and result buttons are darker green.

Table describing layout

Display 1
Display 2
help logarithm root
7 8 9 power factorial
4 5 6 multiplication division
1 2 3 addition subtraction
0 . clear result

Keyboard mapping

Every button can be substituated with key on keyboard.

"?/h/H" - help

"+" addition
"-" subtraction
"*" multiplication
"/" division
"p/P" power
"r/R" root
"f/F" factorial
"l/L" logarithm

"c/C" - clear
"return/enter" - result

Number are of course number keys.

How to use

Start the aplication.

Click on operands and operations you want to use.
You can also use keyboard for that.
On Display 1 you will see what you have written.
When you want to see the result click on button "=" or use "return/enter" key.
Result will show on Display 2.

To exit the aplication hit the red cross in top right corner.
You can also do "alt + f4"
